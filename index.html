기기 간 동기화 (GitHub Gist)

노트 내용과 퀴즈 진도(복습 스케줄·통계)가 함께 동기화됩니다. PC에서 편집 후 업로드, 휴대폰에서는 같은 Gist ID를 넣고 가져오기만 하면 됩니다. 휴대폰에도 토큰을 넣으면 퀴즈를 마칠 때마다 진도가 자동 업로드됩니다. (토큰: github.com → Settings → Developer settings → Personal access tokens, gist 권한)

이 노드부터 DFS 순서로 조립한 답안 초안입니다.

    🏛 변호사시험 기출 불러오기
    
법무부가 공개한 변호사시험 선택형 기출입니다(공공누리 제1유형 · 출처: 법무부). 위에서 선택한 항목에 연결되며, 이미 불러온 문제는 중복 추가되지 않습니다.

    
    

    

  

  

    📚 문제집에서 일괄 가져오기
    
① 변환 프롬프트 복사 → ② AI 채팅(ChatGPT·Claude 등)에 프롬프트와 문제집 내용을 붙여넣기 → ③ 나온 JSON을 아래에 붙여넣고 가져오기. 가져온 문제는 위에서 선택한 항목에 연결됩니다. 개인 학습 목적으로만 사용하세요.
